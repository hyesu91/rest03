# 개발일지 — rest03 건설사 회사 사이트 템플릿

- **작업일**: 2026-06-10
- **저장소**: https://github.com/hyesu91/rest03
- **배포 URL**: https://hyesu91.github.io/rest03/

---

## 1. 목표

진흥기업(chinhung.co.kr)에서 제공받은 페이지 소스(메인 / 지속가능경영-품질경영)를
참고하여, **건설사 회사 홈페이지를 React 템플릿으로 재구성**한다.
모든 이미지는 운영 시 교체 가능한 **플레이스홀더**로 표기한다.

## 2. 기술 선택

| 항목 | 선택 | 사유 |
| --- | --- | --- |
| 프레임워크 | React 18 + Vite 5 | 빠른 HMR, 가벼운 정적 빌드 |
| 라우팅 | React Router 6 (**HashRouter**) | GitHub Pages 정적 호스팅에서 새로고침 404 방지 |
| 스타일 | Tailwind CSS 3 | 유틸리티 기반으로 소스 레이아웃을 빠르게 재현 |
| 폰트 | Pretendard (CDN) | 원본 사이트와 동일 계열 한글 폰트 |
| 배포 | GitHub Actions → Pages | main push 시 자동 빌드·배포 |
| base 경로 | `/rest03/` | 리포명 기반 Pages 경로 |

## 3. 제공 소스 → 구현 매핑

작업의 핵심은 **제공된 HTML 소스를 영역별로 구분**하여 React 컴포넌트로 옮긴 것이다.

## 4. 구조 설계

```
src/
├── data/site.js          # 회사정보·GNB·사업·실적·배너·IR 데이터 일원화
├── components/
│   ├── Header.jsx         # GNB (외부링크 분기 SubLink)
│   ├── Footer.jsx
│   ├── SubPageLayout.jsx  # sticky 탭 + 페이지헤드 오버레이(hero prop)
│   ├── Placeholder.jsx    # 이미지 플레이스홀더 (라벨+비율+사선패턴)
│   ├── ScrollToTop.jsx
│   └── ScrollToTopButton.jsx
└── pages/
    ├── Home.jsx           # 메인 6개 섹션
    ├── Business.jsx       # 카테고리 탭 + 실적 그리드 + 페이지네이션
    ├── Sustainability.jsx # 품질경영 본문 + 기타 탭 공통 본문
    ├── About.jsx          # CEO 인사말 / 연혁 / 비전·브랜드
    └── SimplePage.jsx     # 투자정보·고객센터·인재채용 등
```

**설계 의도**
- `src/data/site.js` **한 파일만 수정**하면 회사명·메뉴·실적·배너가 전부 반영되도록 데이터를 분리 → 템플릿 재사용성 확보.
- `SubPageLayout` 에 `hero` prop 을 추가해 서브페이지마다 페이지헤드 오버레이를 데이터로 주입.
- `Placeholder` 로 이미지 자리를 명확히 표기 → 실제 운영 시 `<img>` 로 교체만 하면 됨.
